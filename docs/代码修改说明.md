### 2. 修改现有函数

#### 修改：`import_srt_to_timeline()` 函数
**位置：** 第 837-891 行
**修改方式：** 完全替换

**原函数签名：**
```python
def import_srt_to_timeline(srt_path):
```

**新函数签名：**
```python
def import_srt_to_timeline(srt_path, insert_position_frame=0):
```

**主要变化：**
1. 添加了 `insert_position_frame` 参数（默认值为 0，保持向后兼容）
2. 删除了"删除所有字幕"的逻辑
3. 添加了时间偏移调整逻辑
4. 添加了增量删除重叠字幕的逻辑

**完整的新函数代码见 `subtitle_functions_new.py`**

## 常见问题排查

### 问题 1：字幕时间仍然不对
**原因：** 调用时没有传递 `insert_position_frame` 参数
**解决：** 检查所有调用位置，确保传递了正确的 `start_frame`

### 问题 2：所有字幕仍然被删除
**原因：** 使用了旧版本的函数
**解决：** 确认已经替换了 `import_srt_to_timeline` 函数

### 问题 3：找不到 `adjust_srt_timestamps` 函数
**原因：** 新函数没有添加到文件中
**解决：** 按步骤 3 添加三个新函数

### 问题 4：字幕文件导入失败
**原因：** SRT 文件路径或格式问题
**解决：** 检查控制台输出，查看具体错误信息
