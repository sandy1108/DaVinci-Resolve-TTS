## 代码中的参数映射

### synthesize() 方法参数映射

```python
# 项目代码参数 → API 参数
text              → payload["text"]
model             → payload["model"]
voice_id          → payload["voice_setting"]["voice_id"]
speed             → payload["voice_setting"]["speed"]
vol               → payload["voice_setting"]["vol"]
pitch             → payload["voice_setting"]["pitch"]
emotion           → payload["voice_setting"]["emotion"]  # 如果不是"默认"
file_format       → payload["audio_setting"]["format"]
subtitle_enable   → payload["subtitle_enable"]
sound_effects     → payload["voice_modify"]["sound_effects"]  # 如果不是"默认"

# 硬编码的参数
stream            → False
sample_rate       → 32000
bitrate           → 128000
channel           → 2
```

### submit_clone_job() 方法参数映射

```python
# 项目代码参数 → API 参数
file_id           → payload["file_id"]
voice_id          → payload["voice_id"]
need_nr           → payload["need_noise_reduction"]
need_vn           → payload["need_volume_normalization"]
text              → payload["text"]  # 可选
                  → payload["model"] = "speech-02-hd"  # 如果提供text
```

## 建议的参数优先级

### 🔴 高优先级（建议立即添加）

1. **language_boost** - 多语种支持
   - 用途：提升非中文语言的识别和合成质量
   - 实现难度：低
   - 影响范围：国际化场景

2. **output_format="url"** - URL输出
   - 用途：避免大文件 hex 编码传输
   - 实现难度：低
   - 影响范围：性能优化

3. **text_normalization** - 文本规范化
   - 用途：改善数字、日期等的朗读
   - 实现难度：低
   - 影响范围：专业内容

### 🟡 中优先级（可选增强）

4. **clone_prompt** - 示例音频
   - 用途：提升音色复刻质量
   - 实现难度：中
   - 影响范围：音色复刻功能

5. **pronunciation_dict** - 发音字典
   - 用途：解决多音字、专有名词发音
   - 实现难度：中
   - 影响范围：中文内容质量

6. **voice_modify 完整参数** - 精细音色调整
   - 用途：更灵活的音色控制
   - 实现难度：低
   - 影响范围：高级用户

### 🟢 低优先级（特殊场景）

7. **timber_weights** - 混合音色
   - 用途：创造独特音色
   - 实现难度：高
   - 影响范围：创意场景

8. **latex_read** - LaTeX 支持
   - 用途：科技、教育内容
   - 实现难度：低
   - 影响范围：特定领域

9. **aigc_watermark** - 水印
   - 用途：版权保护
   - 实现难度：低
   - 影响范围：商业场景
