# 字幕增量替换问题分析与解决方案

## 问题描述

**当前行为：**
每次进行 TTS 转换并生成字幕后，新的字幕文件会**完全替换**整个视频的所有字幕，导致之前的字幕内容全部丢失。

**期望行为：**
1. **增量替换**：只替换当前播放头位置的字幕，保留其他位置的字幕
2. **多轨道插入**：如果无法增量替换，至少可以将新字幕插入到不同的字幕轨道

---

## 问题根源分析

### 当前代码实现（第 843-890 行）

```python
def import_srt_to_timeline(srt_path):
    """
    将指定 .srt 文件导入并追加到当前时间线。
    返回 True 表示成功，False 表示失败。
    """
    # 1. 获取当前项目和时间线
    timeline = current_project.GetCurrentTimeline()
    
    # 2. ⚠️ 问题所在：删除所有字幕轨道中的所有片段
    sub_count = timeline.GetTrackCount("subtitle")
    for ti in range(1, sub_count + 1):
        items = timeline.GetItemListInTrack("subtitle", ti)
        if items:
            timeline.DeleteClips(items)  # ← 这里删除了所有字幕！
    
    # 3. 导入新的 .srt 文件到媒体池
    imported = media_pool.ImportMedia([srt_path])
    
    # 4. 将字幕追加到时间线
    new_clip = imported[0]
    success = media_pool.AppendToTimeline([new_clip])
    
    return True
```

### 问题分析

**第 2 步的问题：**
```python
# 遍历所有字幕轨道
for ti in range(1, sub_count + 1):
    items = timeline.GetItemListInTrack("subtitle", ti)
    if items:
        timeline.DeleteClips(items)  # ← 删除该轨道的所有字幕片段
```

这段代码会：
1. 遍历所有字幕轨道（1 到 sub_count）
2. 获取每个轨道上的所有字幕片段
3. **无条件删除所有字幕片段**

**结果：** 每次导入新字幕时，整个时间线的所有字幕都被清空了。

## 技术可行性分析

### 方案 1：增量替换（精确替换当前位置的字幕）

#### ✅ 技术可行性：**可行**

**实现思路：**
1. 获取当前播放头位置（`timeline.GetCurrentTimeCode()`）
2. 解析新的 SRT 文件，获取字幕的时间范围
3. 只删除与新字幕时间范围**重叠**的旧字幕片段
4. 插入新字幕到指定位置

**优点：**
- ✅ 保留其他位置的字幕
- ✅ 精确控制替换范围
- ✅ 符合用户预期

**缺点：**
- ⚠️ 需要解析 SRT 文件获取时间信息
- ⚠️ 需要计算时间重叠逻辑
- ⚠️ 实现相对复杂

**DaVinci Resolve API 支持：**
```python
# ✅ 支持获取字幕片段的时间范围
item.GetStart()      # 获取起始帧
item.GetEnd()        # 获取结束帧
item.GetDuration()   # 获取持续时间

# ✅ 支持删除特定片段
timeline.DeleteClips([item1, item2, ...])

# ✅ 支持在指定位置插入字幕
media_pool.AppendToTimeline([clip_info])
```

#### 实现步骤：

**步骤 1：解析 SRT 文件获取时间范围**
```python
def parse_srt_time_range(srt_path, frame_rate):
    """解析 SRT 文件，返回 (start_frame, end_frame)"""
    with open(srt_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 解析第一个和最后一个时间戳
    # SRT 格式：00:00:01,000 --> 00:00:03,000
    import re
    timestamps = re.findall(r'(\d{2}:\d{2}:\d{2},\d{3})', content)
    
    if len(timestamps) >= 2:
        start_tc = timestamps[0]
        end_tc = timestamps[-1]
        
        start_frame = timecode_to_frames(start_tc, frame_rate)
        end_frame = timecode_to_frames(end_tc, frame_rate)
        
        return start_frame, end_frame
    
    return None, None
```

**步骤 2：只删除重叠的字幕片段**
```python
def delete_overlapping_subtitles(timeline, start_frame, end_frame):
    """删除与指定时间范围重叠的字幕片段"""
    sub_count = timeline.GetTrackCount("subtitle")
    
    for track_index in range(1, sub_count + 1):
        items = timeline.GetItemListInTrack("subtitle", track_index)
        
        if items:
            overlapping_items = []
            
            for item in items:
                item_start = item.GetStart()
                item_end = item.GetEnd()
                
                # 检查是否重叠
                if not (item_end < start_frame or item_start > end_frame):
                    overlapping_items.append(item)
            
            # 只删除重叠的片段
            if overlapping_items:
                timeline.DeleteClips(overlapping_items)
                print(f"删除了 {len(overlapping_items)} 个重叠的字幕片段")
```

**步骤 3：修改 import_srt_to_timeline 函数**
```python
def import_srt_to_timeline_incremental(srt_path):
    """增量导入字幕，只替换重叠部分"""
    timeline = current_project.GetCurrentTimeline()
    frame_rate = float(timeline.GetSetting("timelineFrameRate"))
    
    # 1. 解析 SRT 文件的时间范围
    start_frame, end_frame = parse_srt_time_range(srt_path, frame_rate)
    
    if start_frame is None:
        print("无法解析 SRT 时间范围")
        return False
    
    # 2. 只删除重叠的字幕片段
    delete_overlapping_subtitles(timeline, start_frame, end_frame)
    
    # 3. 导入新字幕
    media_pool = current_project.GetMediaPool()
    imported = media_pool.ImportMedia([srt_path])
    
    if not imported:
        return False
    
    # 4. 添加到时间线
    success = media_pool.AppendToTimeline([imported[0]])
    
    return success
```

### 方案 2：多轨道插入（每次插入到新轨道）

#### ✅ 技术可行性：**可行且更简单**

**实现思路：**
1. 不删除任何现有字幕
2. 每次导入字幕时，找到第一个空的字幕轨道
3. 如果没有空轨道，创建新的字幕轨道
4. 将新字幕插入到该轨道

**优点：**
- ✅ 实现简单
- ✅ 保留所有历史字幕
- ✅ 用户可以手动管理轨道
- ✅ 不需要解析 SRT 文件

**缺点：**
- ⚠️ 轨道数量会不断增加
- ⚠️ 需要用户手动清理不需要的轨道
- ⚠️ 多个字幕可能同时显示（需要手动禁用轨道）

**DaVinci Resolve API 支持：**
```python
# ✅ 支持获取字幕轨道数量
timeline.GetTrackCount("subtitle")

# ✅ 支持添加新轨道
timeline.AddTrack("subtitle")

# ✅ 支持检查轨道是否为空
timeline.GetItemListInTrack("subtitle", track_index)
```

#### 实现步骤：

**步骤 1：找到第一个空的字幕轨道**
```python
def get_first_empty_subtitle_track(timeline):
    """获取第一个空的字幕轨道索引，如果没有则创建新轨道"""
    sub_count = timeline.GetTrackCount("subtitle")
    
    # 检查现有轨道
    for track_index in range(1, sub_count + 1):
        items = timeline.GetItemListInTrack("subtitle", track_index)
        if not items:  # 轨道为空
            return track_index
    
    # 所有轨道都有内容，创建新轨道
    if timeline.AddTrack("subtitle"):
        return sub_count + 1
    
    return None
```

**步骤 2：修改 import_srt_to_timeline 函数**
```python
def import_srt_to_timeline_multitrack(srt_path):
    """将字幕导入到空的字幕轨道，不删除现有字幕"""
    timeline = current_project.GetCurrentTimeline()
    
    # 1. 找到空的字幕轨道（或创建新轨道）
    target_track = get_first_empty_subtitle_track(timeline)
    
    if target_track is None:
        print("无法创建新的字幕轨道")
        return False
    
    print(f"将字幕导入到轨道 {target_track}")
    
    # 2. 导入字幕到媒体池
    media_pool = current_project.GetMediaPool()
    imported = media_pool.ImportMedia([srt_path])
    
    if not imported:
        return False
    
    # 3. 添加到指定轨道
    # 注意：AppendToTimeline 会自动选择轨道，可能需要额外处理
    success = media_pool.AppendToTimeline([imported[0]])
    
    return success
```

**注意事项：**
DaVinci Resolve 的 `AppendToTimeline` 方法可能不支持直接指定字幕轨道。如果不支持，可以考虑：
1. 先导入到默认轨道
2. 然后使用 API 移动字幕片段到目标轨道（如果 API 支持）

### 方案 3：混合方案（推荐）

#### ✅ 技术可行性：**可行且最佳**

**实现思路：**
结合方案 1 和方案 2 的优点：
1. 提供**两种模式**供用户选择：
   - **替换模式**：增量替换重叠的字幕（方案 1）
   - **追加模式**：插入到新轨道（方案 2）
2. 在 UI 中添加选项让用户选择模式

**优点：**
- ✅ 灵活性最高
- ✅ 满足不同使用场景
- ✅ 用户可以根据需求选择

**实现：**
```python
def import_srt_to_timeline(srt_path, mode="replace"):
    """
    导入字幕到时间线
    
    Args:
        srt_path: SRT 文件路径
        mode: "replace" (替换重叠) 或 "append" (追加到新轨道)
    """
    if mode == "replace":
        return import_srt_to_timeline_incremental(srt_path)
    elif mode == "append":
        return import_srt_to_timeline_multitrack(srt_path)
    else:
        return import_srt_to_timeline_incremental(srt_path)
```
