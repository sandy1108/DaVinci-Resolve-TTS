# 字幕插入位置问题分析与解决方案

## 问题描述

**现象：**
- ✅ 音频文件正确插入到光标位置
- ❌ 字幕文件插入到了字幕内容的结尾位置，而不是光标位置

**示例：**
```
光标位置：00:01:30 (2700帧)
音频插入：00:01:30 ✅ 正确
字幕插入：00:01:35 ❌ 错误（应该在 00:01:30）
```

---

## 问题根源分析

### DaVinci Resolve API 的行为差异

#### 音频插入（正确）
```python
# add_to_media_pool_and_timeline 函数
clip_info = {
    "mediaPoolItem": selected_clip,
    "startFrame": 0,
    "endFrame": clip_duration_frames - 1,
    "trackIndex": track_index,
    "recordFrame": start_frame,  # ← 指定插入位置
    "stereoEye": "both"  
}

timeline_item = media_pool.AppendToTimeline([clip_info])
```

**关键：** 使用 `clipInfo` 字典，可以通过 `recordFrame` 参数**精确指定插入位置**。

---

#### 字幕插入（错误）
```python
# import_srt_to_timeline 函数（当前实现）
imported = media_pool.ImportMedia([srt_path_to_import])
new_clip = imported[0]
success = media_pool.AppendToTimeline([new_clip])  # ← 没有指定位置
```

**问题：**
1. 直接传递 `MediaPoolItem` 对象，而不是 `clipInfo` 字典
2. **没有使用 `recordFrame` 参数指定插入位置**
3. DaVinci Resolve 使用 SRT 文件中的时间戳作为插入位置

---

### 为什么会插入到字幕结尾？

**SRT 文件内容示例：**
```srt
1
00:00:00,000 --> 00:00:02,000
第一句话

2
00:00:02,000 --> 00:00:05,000
第二句话
```

**调整后的 SRT 文件（偏移 90 秒）：**
```srt
1
00:01:30,000 --> 00:01:32,000
第一句话

2
00:01:32,000 --> 00:01:35,000
第二句话
```

**DaVinci Resolve 的行为：**
- 读取 SRT 文件中的时间戳
- 将字幕片段放置在对应的时间位置
- **但是 `AppendToTimeline` 可能会将播放头移动到字幕的结束位置**
- 或者 API 的默认行为是追加到时间线末尾

---

## 解决方案

### 方案 1：使用 clipInfo 字典（推荐）

**核心思路：** 像音频一样，使用 `clipInfo` 字典指定字幕的插入位置。

#### 修改后的代码

```python
def import_srt_to_timeline(srt_path, insert_position_frame=0):
    """
    将字幕导入到时间线，支持时间偏移和增量替换
    """
    # ... 前面的代码保持不变 ...
    
    # 5. 导入字幕到媒体池
    media_pool = current_project.GetMediaPool()
    root_folder = media_pool.GetRootFolder()
    media_pool.SetCurrentFolder(root_folder)

    # 删除媒体池中同名旧条目
    file_name = os.path.basename(srt_path_to_import)
    for clip in root_folder.GetClipList():
        if clip.GetName() == file_name:
            media_pool.DeleteClips([clip])
            break

    imported = media_pool.ImportMedia([srt_path_to_import])
    if not imported:
        print(f"错误：字幕文件导入失败 -> {srt_path_to_import}")
        return False

    # 6. 获取字幕片段信息
    subtitle_clip = imported[0]
    
    # 获取字幕的持续时间（帧数）
    # 注意：字幕文件可能没有 Duration 属性，需要从 SRT 解析
    subtitle_duration_frames = end_frame - start_frame
    
    # 7. 创建 clipInfo 字典，指定插入位置
    clip_info = {
        "mediaPoolItem": subtitle_clip,
        "startFrame": 0,  # 字幕片段的起始帧（相对于片段本身）
        "endFrame": subtitle_duration_frames - 1,  # 字幕片段的结束帧
        "recordFrame": start_frame,  # ← 关键：指定在时间线上的插入位置
    }
    
    # 8. 将字幕添加到时间线
    success = media_pool.AppendToTimeline([clip_info])
    
    if not success:
        print("错误：将字幕添加到时间线失败")
        return False

    print(f"字幕已成功加载到时间线: {file_name}，位置: {start_frame} 帧")
    return True
```

**关键变化：**
1. 使用 `clip_info` 字典代替直接传递 `MediaPoolItem`
2. 添加 `recordFrame` 参数，指定插入位置为 `start_frame`
3. 使用解析出的 `start_frame` 和 `end_frame` 计算持续时间

---

### 方案 2：不调整 SRT 时间戳，使用 recordFrame 偏移

**核心思路：** 不修改 SRT 文件的时间戳，而是通过 `recordFrame` 参数实现偏移。

#### 优点
- 不需要创建临时文件
- 更简洁

#### 缺点
- 如果 DaVinci Resolve 同时使用 SRT 时间戳和 recordFrame，可能会冲突

#### 代码示例

```python
def import_srt_to_timeline(srt_path, insert_position_frame=0):
    """
    将字幕导入到时间线，使用 recordFrame 实现偏移
    """
    # 1. 获取时间线
    project_manager = resolve.GetProjectManager()
    current_project = project_manager.GetCurrentProject()
    timeline = current_project.GetCurrentTimeline()
    frame_rate = float(timeline.GetSetting("timelineFrameRate"))
    
    # 2. 解析原始 SRT 的时间范围（不调整）
    start_frame_relative, end_frame_relative = parse_srt_time_range(srt_path, frame_rate)
    
    if start_frame_relative is None:
        print("无法解析 SRT 时间范围")
        return False
    
    # 3. 计算实际的时间范围（加上偏移）
    actual_start_frame = insert_position_frame + start_frame_relative
    actual_end_frame = insert_position_frame + end_frame_relative
    
    print(f"字幕时间范围：{actual_start_frame} - {actual_end_frame} 帧")
    
    # 4. 删除重叠的旧字幕
    delete_overlapping_subtitles(timeline, actual_start_frame, actual_end_frame)
    
    # 5. 导入字幕到媒体池
    media_pool = current_project.GetMediaPool()
    imported = media_pool.ImportMedia([srt_path])
    
    if not imported:
        return False
    
    subtitle_clip = imported[0]
    
    # 6. 使用 clipInfo 指定插入位置
    clip_info = {
        "mediaPoolItem": subtitle_clip,
        "startFrame": start_frame_relative,  # SRT 中的起始时间
        "endFrame": end_frame_relative,      # SRT 中的结束时间
        "recordFrame": insert_position_frame,  # ← 时间线上的插入位置
    }
    
    # 7. 添加到时间线
    success = media_pool.AppendToTimeline([clip_info])
    
    return success
```

## 推荐的完整解决方案

### 修改后的 import_srt_to_timeline 函数

```python
def import_srt_to_timeline(srt_path, insert_position_frame=0):
    """
    将字幕导入到时间线，支持时间偏移和增量替换
    
    Args:
        srt_path: SRT 文件路径
        insert_position_frame: 音频插入位置（帧数），默认为0
    
    Returns:
        True 表示成功，False 表示失败
    """
    # 1. 获取 Resolve、ProjectManager、Project、Timeline
    project_manager = resolve.GetProjectManager()
    current_project = project_manager.GetCurrentProject()
    if current_project is None:
        print("错误：未找到当前项目")
        return False

    timeline = current_project.GetCurrentTimeline()
    if timeline is None:
        print("错误：未找到当前时间线")
        return False
    
    frame_rate = float(timeline.GetSetting("timelineFrameRate"))
    
    # 2. 解析原始 SRT 的时间范围（不调整时间戳）
    start_frame_relative, end_frame_relative = parse_srt_time_range(srt_path, frame_rate)
    
    if start_frame_relative is None:
        print("无法解析 SRT 时间范围")
        return False
    
    # 3. 计算实际的时间范围（加上偏移）
    actual_start_frame = insert_position_frame + start_frame_relative
    actual_end_frame = insert_position_frame + end_frame_relative
    
    print(f"字幕相对时间：{start_frame_relative} - {end_frame_relative} 帧")
    print(f"字幕实际位置：{actual_start_frame} - {actual_end_frame} 帧")
    
    # 4. 删除重叠的旧字幕（使用实际位置）
    delete_overlapping_subtitles(timeline, actual_start_frame, actual_end_frame)
    
    # 5. 导入字幕到媒体池
    media_pool = current_project.GetMediaPool()
    root_folder = media_pool.GetRootFolder()
    media_pool.SetCurrentFolder(root_folder)

    # 删除媒体池中同名旧条目，避免重复
    file_name = os.path.basename(srt_path)
    for clip in root_folder.GetClipList():
        if clip.GetName() == file_name:
            media_pool.DeleteClips([clip])
            break

    imported = media_pool.ImportMedia([srt_path])
    if not imported:
        print(f"错误：字幕文件导入失败 -> {srt_path}")
        return False

    # 6. 获取字幕片段
    subtitle_clip = imported[0]
    
    # 7. 创建 clipInfo 字典，指定插入位置
    # 关键：使用 recordFrame 指定时间线上的位置
    clip_info = {
        "mediaPoolItem": subtitle_clip,
        "startFrame": start_frame_relative,      # 字幕内容的起始帧（相对）
        "endFrame": end_frame_relative,          # 字幕内容的结束帧（相对）
        "recordFrame": insert_position_frame,    # ← 时间线上的插入位置（绝对）
    }
    
    # 8. 将字幕添加到时间线
    success = media_pool.AppendToTimeline([clip_info])
    
    if not success:
        print("错误：将字幕添加到时间线失败")
        return False

    print(f"字幕已成功加载到时间线: {file_name}，位置: {insert_position_frame} 帧")
    return True
```
